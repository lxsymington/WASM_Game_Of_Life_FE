This page is a continuation of the learning exercise from the Rust/WASM book. The Rust/WASM book can be found here.

The graph behind is an implementation of Conway's Game Of Life. In this implementation it is possible to toggle a cell by clicking on it. A Glider can be inserted by holding down the Ctrl key while clicking and a Pulsar while holding down the Shift key.

Different widths, heights and cell counts will produce different results (so try changing them). Some conditions will produce stable structures so nothing will happen, others will collapse immediately (but this can be interesting as a sandpit for playing with insertions).
